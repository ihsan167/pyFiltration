Calculator Basis

P-CADR Particle clean air delivery from airflow, particle efficiency, bypass leakage, and room mixing.
F-CADR Formaldehyde clean air delivery from airflow, gas efficiency, bypass leakage, and room mixing.
Filter Size The UI accepts filter dimensions in mm. Frontal area is width x height. Media area can come from a multiplier, pleat geometry, or a known unfolded media area.
Pressure Filter pressure drop is scaled from reference velocity and checked against the fan curve.

Output Meaning

Required Minimum CADR target. It can come from room targets or from direct P-CADR and F-CADR entries.
Clean Estimated performance with a new filter. Pressure drop is lower, so airflow and CADR are usually higher.
Loaded Estimated end-of-life performance with a dirty filter. Use this conservative value for design decisions.
